   II. <var>f</var><sub>2</sub> = (<var>x</var> - 2)<sup>3</sup> − 1,
   III. <var>f</var><sub>3</sub> = 3 / <var>x</var>,  (<var>x</var> > 0).

3. **Метод приближенного решения уравнений:** метод касательных (Ньютона).
